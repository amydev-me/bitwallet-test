
## Common Setup
```bash
npm install
```
copy env file 
```
cp .env.example .env
```
Setup .env file
```
PORT = 3000
DB_USERNAME=USERNAME
DB_PASSWORD=SECRET
DB_DATABASE= DATABASE
DB_HOST =127.0.0.1
```

## Run

```bash
npm run dev
```
